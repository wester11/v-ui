# Security Policy

## Supported versions

Only the latest state of `main` is considered supported for security fixes.

## Reporting a vulnerability

Please do not open public issues for vulnerabilities.

Use GitHub Security Advisories:

- [Report a vulnerability](https://github.com/wester11/v-ui/security/advisories/new)

Include:

- impact
- reproduction steps
- affected components (`backend`, `agent`, `frontend`, `scripts`)

We will acknowledge and triage as soon as possible.
